# monea 美容コスメオンラインサイト作成指示プロンプト

VSCodeで、Next.js App Routerを使って美容・コスメのオンラインサイト「monea」を作成してください。

## ブランド情報

- サイトURL想定：https://moneaglow.com
- ブランド名：monea
- キャッチコピー：自分らしく美しくなりたいすべての人へ
- サブコピー：誰かになるためではなく、自分自身の魅力を見つけ、花開かせるために

## デザイン要件

- CHANEL公式サイトのような、高級感、余白、大きな実写ビジュアル、中央寄せナビ、カテゴリ導線、エディトリアル型の商品セクション構成にする
- CHANELの画像、ロゴ、文言、ブランド資産は一切使わない
- 画像はすべて実写写真にする
- 全体は白、黒、淡いベージュを基調にする
- フォントは細めで上品にする
- 余白を広く取り、情報を詰め込みすぎない
- PC、タブレット、スマホにレスポンシブ最適化する
- スマホではヘッダーナビを横スクロール可能にする
- スマホではカテゴリ、商品、診断選択肢を1カラムにする
- スマホではボタンと入力欄を押しやすいサイズにする

## 演出要件

- メインビジュアルは、白黒の実写写真が徐々に色づいていく演出にする
- 各カテゴリセクションも、ホバーまたはタッチしたら白黒写真が色づく演出にする
- カテゴリをクリックすると、そのカテゴリの下層ページ表示に切り替わるようにする
- 「誰も見たことがない白黒写真に色がついていく」印象を最重要演出として作る
- CSSだけで白黒から色づく演出を実装する

## サイト構成

### 1. ヘッダー

- moneaロゴ
- SKINCARE
- MAKEUP
- FRAGRANCE
- INNER GLOW
- DIAGNOSIS
- MEMBER

### 2. メインビジュアル

表示テキスト：

- monea
- 自分らしく美しくなりたいすべての人へ
- 誰かになるためではなく、自分自身の魅力を見つけ、花開かせるために

CTA：

- 診断をはじめる
- メンバー登録

### 3. メンバー訴求

- 送料無料の表記は入れない
- サンプル進呈の表記は入れない
- 「メンバーはいつでも10%OFF」
- 「メンバー登録」導線を強く見せる

### 4. カテゴリセクション

カテゴリ名と説明文を以下で揃える：

- SKINCARE：肌本来の透明感を育てるケア
- MAKEUP：自分らしさを引き出す色と質感
- FRAGRANCE：記憶に残る印象をまとう香り
- INNER GLOW：内側から整える美容習慣

### 5. 商品セクション

- 商品は固定ではなく、自動的に変わる仕組みにする
- APIから商品一覧を取得し、一定時間ごとにメイン表示商品が切り替わる
- 主要ドロップシッピングAPIに接続できるように、Shopify、CJdropshipping、汎用APIのアダプター構造を作る
- API未設定時はモック商品を表示する

### 6. 診断セクション

- 「無料カウンセリング」ではなく、5分程度でできる簡易診断にする
- MBTIのような雰囲気だが、もっと簡単にする
- 6問程度の選択式診断にする
- 診断結果に応じて、スキンケア、メイク、香り、インナーケアの商品提案を表示する

### 7. ポップアップバナー

- 商品訴求ではなく、メンバー登録を訴求する
- 「メンバーはいつでも10%OFF」
- ボタンは「メンバー登録」

## 不要なもの

以下は現時点では実装しない：

- 利用規約
- 特定商取引法表記
- プライバシーポリシー
- 住所
- 送料無料表記
- サンプル進呈

## 実装要件

- Next.js App Routerで実装
- JavaScriptで実装
- Tailwind CSSは使わない
- app/page.js、app/globals.css、app/layout.js、app/api/products/route.js、lib/productProviders.js を作成する
- .env.local.example を作成する
- npm run dev で動く状態にする
- 画像URLは仮で実写写真を使い、後で差し替え可能にする
