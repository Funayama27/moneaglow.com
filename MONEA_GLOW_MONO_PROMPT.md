# MONEA GLOW｜配色・デザイン指示（白背景モノトーン）（Claude Code 用）

このファイルの本文を Claude Code へそのまま貼ってください。
`monea-glow-mono.html` を**完成版のトップデザイン**として扱い、これに合わせてサイト全体を作ること。新しく配色を考えず、このファイルの見た目・配色・ヒーロー演出をそのまま基準にする。

---

## 完成見本
- `monea-glow-mono.html` がトップページの完成形（白背景モノトーン＋ヒーロー演出）。
- これを下敷きにして、他ページ（about / products / product / contact / law / privacy）も同じ配色・トーンで作る。

## 配色（白基調モノトーン・無彩色のみ。金や暖色は使わない）
- 背景: #FFFFFF ／ オフホワイト #F7F7F5
- 文字: #1A1A1A（黒）／ 補助 #777775（グレー）
- 罫線: rgba(0,0,0,.1)
- 面（カード）: #FFFFFF
- メリハリ用の黒地セクション: #1A1A1A（コンセプト帯・フッター）
- 写真・商品画像: すべてグレースケール（白黒）

## ヒーロー
- `monea-glow-mono.html` のヒーロー（白黒ポートレート＋白銀の光＋光の粒＋ブランド名 MONEA GLOW）を、**改変せずそのまま**使う。
- 画像の data URI もそのまま保持する。光の色（白銀）・速度・ブランド名の位置を変えない。

## テイスト
- 白黒の無彩色、余白広め、Cormorant Garamond の上品なセリフ。ミニマルで高級感。
- 装飾より「余白・タイポ・白黒のコントラスト・光」で見せる。

## 厳守事項
- 金・暖色・その他の有彩色を加えない（モノトーンを保つ）。
- `monea-glow-mono.html` の配色・ヒーロー演出を勝手に変えない。
- 既存の構成・機能・ブランド名・タグライン・コンテンツは変更しない。
- 完了後、トップと各ページが `monea-glow-mono.html` と同じトーンになっていることを確認して報告する。
