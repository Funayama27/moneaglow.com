# MONEA GLOW｜追加指示：人気美容アイテムの自動入荷・自動入れ替え（Claude Code 用）

このファイルの本文を、サイト本体（`BUILD_PROMPT.md`）の実装後に Claude Code へ貼ってください。
作業ディレクトリ `~/github/v0-monea`。参考実装として `monea-glow-trending.html` を同フォルダに置いておくこと。

---

MONEA GLOW に「人気の美容アイテムが自動で入荷・自動で入れ替わる」仕組みを追加実装してください。既存の MONEA GLOW デザイン・ブランド・タグラインは変更しないこと。

## 目的
立ち上げ直後で自店の売上がなくても、外部のドロップシッピング／人気美容トレンドのフィードから人気アイテムを自動取得してトップに並べ、人気度に応じて自動で入れ替える。在庫は持たず、注文が入ったらサプライヤーが直送する。

## 並び順の基準（重要）
- 「自店の売上に連動」ではない。新規ショップに売上はない。
- 並び・入れ替えの基準は、外部フィード由来の人気度（トレンド指数／サプライヤーの人気ランキング）とする。

## 設計ルール（化粧品EC特有・必ず守る）
- 商品を2系統に分ける:
  - 化粧品 = 薬機法の対象。国内発送。成分・製造販売業者名・内容量・使用上の注意の表示が必須。
  - 美容雑貨 = 薬機法の対象外（美顔器・ブラシ・スパ小物等）。海外直送／取り寄せ可。
- 自動入れ替えの主対象は、まず薬機法対象外の「美容雑貨」にする。
- 化粧品をドロップシッピングする場合は、海外直送ではなく国内在庫・正規仕入れに限定する。
- ブランド・商標・知的財産を侵害する商品は扱わない（正規サプライヤーのみ。Shopify の停止リスク回避）。
- 商品データ構造に表示義務フィールド（`ingredients`, `maker`, `volume`, `caution`）を持たせ、化粧品カテゴリでは必須にする。

## フロント実装（参考: monea-glow-trending.html）
- 「いま人気の美容アイテム」セクションを、外部フィード取得関数の戻り値で描画する（固定配列にしない）。
- データ取得層を1ファイルに分離する（例: `js/feed.js`）。
  - 関数: `fetchTrendingItems(limit)` … 人気度（`trend`）の降順で返す。
  - フラグ `USE_SHOPIFY`（既定 false）で、モックデータ ↔ 本番ソースを切り替えられる構造にする。
- 自動更新: 一定間隔で再取得し、トレンド商品が自動で入れ替わる（デモは5秒、本番は数分〜数時間で設定可）。順位が変動した商品は一瞬ハイライトする。
- 各カードの表示: 順位、在庫種別（「国内発送」／「海外直送・取り寄せ」）、お届け目安（リードタイム）、価格、「カートに入れる」。
- 売上個数などゼロ起算の指標は表示しない。
- 取得失敗時は同梱モックにフォールバックする。

## 本番のデータ取得（USE_SHOPIFY = true 時）
人気アイテムの取得元は設定で切り替えられる形にする（エンドポイント・トークンは `.env` から読み、ハードコードしない）。
1. Shopify Storefront API ＋ 自動コレクション（連携ドロップシッピングアプリが商品を流し込む）
2. ドロップシッピングサプライヤーのトレンド／ベストセラーAPI（直接取得）

Shopify 経由の取得イメージ（自動コレクションをトレンド順で取得）:
```graphql
query Trending($n: Int!) {
  collection(handle: "trending") {
    products(first: $n, sortKey: BEST_SELLING) {
      edges {
        node {
          id title handle availableForSale
          featuredImage { url altText }
          priceRange { minVariantPrice { amount currencyCode } }
        }
      }
    }
  }
}
```

## Shopify 側の運用設定（コードではなく手順。README に追記）
1. 美容雑貨向けにドロップシッピングアプリを導入する（DSers / Spocket / Zendrop / AutoDS 等。化粧品は正規卸の NETde卸 等）。
2. サプライヤーの人気・トレンド商品が自動で取り込まれる設定にする。
3. 「いま人気」用の自動コレクション（ハンドル `trending`）を作成し、フロントがそれを取得して自動で入れ替え表示する。
